# Changelog

All notable changes to Swarm for Claude Code will be documented in this file.

## [0.1.0] - 2025-03-05

### Added

- Initial release
- Parallel Claude Code agents with sidebar UI
- Git worktree isolation for each task
- Real-time agent status tracking (busy/idle/waiting/stopped)
- Session persistence with `--continue` support
- Git operations from context menu (commit, sync, merge)
- Task archiving and restoration
- Configurable models (opus/sonnet/haiku)
- Permission modes (plan/autoEdit/fullAuto)
- Notification sounds when tasks complete
- Preserved files copied to new worktrees (.env, etc.)
